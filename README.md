# 🙋‍♀️ Personal Portfolio Website

I custom-designed v1 of my portfolio, and built with React and Vite to showcase my projects and design approach. The site focuses on bold visual design, strong typography, and a clean, modern user experience.

**Live Demo**: https://aliciabuelow.dev

## Features

- Responsive design (media queries & clamp())
- Scroll-based animations using Motion and useInView
- Typewriter-style welcome element
- Interactive hover effects (offset "drop shadows")
- Clean component-based structure with React
- SEO optimization (meta tags & Open Graph)
- Google Analytics integration
- Custom domain with DNS configuration and SSL

## Tech Stack

- React (Vite)
- JavaScript
- Bootstrap
- CSS3
- Motion
- Vercel (deployment)

## Challenges & Learning

This project was a strong learning experience, particularly in:
- Working with new animation tools (Motion, useInView)
- Translating design into responsive layouts
- Refining spacing, alignment, and accessibility
- Iterating on UI decisions to balance aesthetics with usability

I also gained experience with real-world deployment, including connecting a custom Squarespace domain, configuring DNS, and implementing SEO and analytics.
